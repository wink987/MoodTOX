# MoodTOX

Welcome to **MoodTOX**!

MoodTOX is a project for molecular toxicity prediction and structural interpretation. It supports toxicity prediction based on molecular SMILES, model training, performance evaluation, and visualization analysis.

## Project Overview

MoodTOX is mainly designed for multi-organ toxicity prediction tasks. It can be used to analyze the potential risks of compounds across different toxicity endpoints and provide interpretability based on molecular structural information.
